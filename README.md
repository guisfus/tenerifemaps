# Tenerife Maps

Reconstruccion moderna de `tenerifemapsv2` como aplicacion Vue 3 con Vite, Tailwind CSS y soporte bilingue en espanol e ingles.

## Stack

- Vue 3 + TypeScript
- Vite
- Tailwind CSS v4
- Vue I18n
- Leaflet

## Desarrollo

```bash
npm install
npm run dev
```

## Produccion

```bash
npm run build
```

El build generado queda en `dist/`.
